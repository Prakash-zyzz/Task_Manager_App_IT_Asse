# Task Manager App

A Task Manager Progressive Web App built with React, Vite, and Firebase.

## Features
- Firebase Email/Password Authentication
- Email verification on signup
- Create, update, delete tasks
- Task completion and due dates
- Firestore database
- Offline support (PWA)
- Responsive UI

## Tech Stack
- React + Vite
- Firebase Auth & Firestore
- Tailwind CSS
- Framer Motion

## Setup
1. Install dependencies  
   npm install

2. Create `.env` file with Firebase keys  
3. Run the app  
   npm run dev
